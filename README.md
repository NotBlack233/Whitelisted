# Whitelisted

A velocity whitelist plugin.

## Features

### HTTP API

You can access the API from http://127.0.0.1:9876 by default.

> [!TIP]
> Try out our official Python client: [WhitelistedClient](https://github.com/NotBlack233/WhitelistedClient)

> [!WARNING]
> It is **NOT** recommended to open this API to the public directly, since the messages are not encrypted.
